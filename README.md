## Running Locally

[Next.js App Router Playground](https://github.com/vercel/app-playground)

1. Install dependencies:

```sh
pnpm install
```

2. Edit `.env.local`:

```
NEXT_PUBLIC_API=

POSTGRES_URL=
POSTGRES_PRISMA_URL=
POSTGRES_URL_NON_POOLING=
POSTGRES_USER=
POSTGRES_HOST=
POSTGRES_PASSWORD=
POSTGRES_DATABASE=
```

3. Start the dev server:

```sh
pnpm dev
```

http://localhost:3000
